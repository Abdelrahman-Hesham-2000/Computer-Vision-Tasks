## 📖 About

This repository contains a collection of Computer Vision and Deep Learning projects implemented using modern AI frameworks such as **PyTorch** and **TensorFlow**.

The goal of this repository is to showcase practical implementations of computer vision algorithms, image processing techniques, and deep learning models across different applications.

---

# 🚀 Projects

| Project | Description | Technologies |
|---------|-------------|--------------|
| 🌊 **Water Segmentation Using U-Net** | Semantic segmentation of water bodies from multispectral satellite images using U-Net. | PyTorch, OpenCV |
| 👁 **Eye Disease Classification** | Multi-class eye disease classification using Convolutional Neural Networks and Transfer Learning. | TensorFlow, CNN, EfficientNet |
